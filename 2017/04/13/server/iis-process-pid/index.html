查看IIS中网站所对应的进程信息

2017-04-13
  • 服务器

导语

有时我们一台服务器会部署多个项目，这时如果要查看某个项目所对应的进程，就会比较麻烦，下面我们已Windows Server 2008（IIS7）来看一下如何查询网站所对应的进程。

Windows Server 2008（IIS7）操作步骤：

  1. 进入CMD
  2. 输入命令： cd C:\Windows\System32\inetsrv 进入inetsrv 目录
  3. 输入命令： appcmd list wp
  4. 任务管理器中点击查看–选择列–选择PID（进程标识符）
