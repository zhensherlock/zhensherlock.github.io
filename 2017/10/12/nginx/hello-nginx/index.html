导语

最近 JetBrains 授权服务器封的比较严重，在 2017.3 之前我们可以使用 License Server 设置成 http://idea.imsxm.com 一旦更新到2017.3之后的版本，就会发现网上很多授权服务器都被禁用了，
但是我们发现使用反向代理到 http://idea.imsxm.com 就可以授权，下面我们来看一下如何反向代理。

安装Nginx

打开 终端，执行 brew install nginx

安装完之后，执行 nginx
