解决 MAC 无法打开应用或应用文件已损坏问题

2017-09-12
  • Mac

导语

Mac安全机制中默认只信任 Mac App Store 下载的软件和拥有开发者 ID 签名的应用程序。
就是说 OS X 默认只能安装苹果审核通过的软件，这也是为了用户不会安装流氓软件中招。
但是当我们在安装网上的破解软件时就会遇到下面这种情况。

解决方案

打开 系统偏好设置 -> 安全性与隐私

选择任何来源选项

如果你没有这个选项，打开 终端，执行 sudo spctl --master-disable，再次重新打开安全性与隐私，就会出现任何来源了。
